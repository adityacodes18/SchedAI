# SchedAI Features

## Project Goal

SchedAI is an AI-powered productivity scheduler that helps users organize tasks, optimize schedules, and improve productivity through intelligent planning.

# AI Features

These features make the project stand out.

## Duration Prediction Agent

Input:

Task: Build Login Page

Output:

Estimated Time: 3 Hours

Purpose:

Estimate task completion time automatically.

---

## Priority Analysis Agent

Analyzes:

- Deadline
- Importance
- Workload

Outputs recommended priority level.

---

## Smart Scheduler Agent

Creates optimized schedules using:

- Available hours
- Task priorities
- Deadlines
- Estimated durations

---

## Auto Rescheduling Agent

When a task is missed:

- Detect incomplete task
- Find next available slot
- Update schedule automatically

---

## Schedule Explanation Agent

Explains AI decisions.

Example:

"This task was scheduled first because it has a high priority and a deadline within 24 hours."
